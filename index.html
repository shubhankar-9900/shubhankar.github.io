I tried all kinds of ways to challenge my brain - including chess and a Rubik’s cube, until I finally found coding.

The first program I built was a simple way to create graphics in Logo using the infamous turtle; it was the most fun I had solving a problem. From then on, I dabbled in several programming languages and ended up creating robust applications while providing maximum focus to optimization and security.

I fell in love with providing unique user experience, and the joy I got when someone genuinely praised my work only propelled me further. Ultimately, I want to build humane software products which will help society at large as well as our environment.
